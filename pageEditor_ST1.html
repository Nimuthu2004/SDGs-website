Splash Page

Technical Description

HTML Techniques:
- Used a <meta> tag to automatically redirect to the Home Page after 4 seconds.
- Structured the content using <div> and <h1> tags for headings and paragraphs.
- Included a button linking to the Page Editor using an onclick event.

CSS Techniques:
- Applied a full-screen background image with an overlay for better readability.
- Used flexbox to center the content vertically and horizontally.
- Added animations (fadeIn) for a smooth transition effect.
- Created a spinning loader using CSS keyframes (spin).

Key Tags and Properties:
- background-image: To set the background image.
- flexbox: For centering content.
- animation: For fade-in and spinning effects.

Accessibility

In splash.html, accessibility was ensured with a clear semantic structure and visual contrast. The background overlay improves text readability, and the button includes hover effects for better interactivity. The loader animation provides feedback during the redirection.

Volunteer Page

Technical Description

HTML Techniques:
- Structured the page using <section> and <div> tags.
- Used <video> tag for a background video.
- Created cards for volunteer opportunities using <div> and <button> tags.

CSS Techniques:
- Implemented a grid layout for the volunteer cards.
- Added hover effects on cards using transform and box-shadow.
- Styled the feedback form and review section with flexbox.

Key Tags and Properties:
- grid-template-columns: For a two-column layout.
- transform: For hover effects.
- flexbox: For aligning form elements.

Accessibility

In volunteer.html, semantic HTML like <header> and <main> aids screen readers. The video background is muted to avoid disruptions, and form elements are labeled for accessibility. Hover effects on cards enhance usability.
